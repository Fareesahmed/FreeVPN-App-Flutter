 <strong>Features👇 : </strong>
<ul>
<li>Many Free Public Servers (From: https://www.vpngate.net/en/)
<li>Secure OpenVPN Protocol
<li>No Signing Required
<li>IP Address Test
<li>Data Caching
<li>Google Test Ads Integrated
<li>Decent UI with New Material Widgets
<li>And Much More....
</ul>
